# Crypto Sentinel

Асинхронный микросервис на FastAPI для мониторинга цен криптовалют с Binance.

## Возможности

- Периодический опрос цен по списку тикеров (из .env).
- Сохранение истории цен в PostgreSQL.
- Проверка порога для BTC (предупреждение в лог и Sentry).
- Асинхронные HTTP-запросы через единый клиент.
- Graceful shutdown с корректным освобождением ресурсов.
- Полное логирование и сбор ошибок в Sentry.
